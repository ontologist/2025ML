Python Basics: Variables | Python基礎: 変数

In programming, a variable is like a labeled box that stores information. You give the box a name, and you can put different types of information inside.

プログラミングでは、変数は情報を保存するラベル付きの箱のようなものです。

Creating Variables in Python

# Variable assignment uses the equals sign (=)
company_name = "ABC Corporation"
employee_count = 500
average_rating = 4.5
is_hiring = True

# You can print variables to see their values
print(company_name)        # Output: ABC Corporation
print(employee_count)     # Output: 500

Variable Naming Rules

Rules You Must Follow:

  • Variable names can contain letters, numbers, and underscores
  • Must start with a letter or underscore (not a number)
  • Case-sensitive: CompanyName ≠ companyname
  • Cannot use Python keywords (like print, if, True)

Best Practices:

  • Use descriptive names: employee_count not ec
  • Use lowercase with underscores: company_name (snake_case)
  • Avoid confusing names: l (lowercase L) looks like 1
# Good variable names
company_name = "Toyota"
employee_count = 370000
founded_year = 1937
is_remote_friendly = True

# Poor variable names (avoid these)
cn = "Toyota"              # Too short, unclear
EmployeeCount = 370000     # Not following convention
x = 1937                   # Meaningless

Python Data Types | Pythonデータ型

Different types of data require different types of storage. Python has several built-in data types. We'll focus on the four most important for our work.

さまざまなタイプのデータには、さまざまなタイプのストレージが必要です。

The Four Essential Data Types

Data Type Description Example Use Case
String (str)
文字列		 Text enclosed in quotes "Technology"
'Remote work'		 Company names, descriptions, industries
Integer (int)
整数		 Whole numbers (no decimals) 500
2010		 Employee counts, founded years
Float
浮動小数点		 Numbers with decimal points 4.5
4500000.00		 Ratings, salaries, percentages
Boolean (bool)
ブール値		 True or False values True
False		 Hiring status, remote work availability

Working with Different Data Types

# Strings - use quotes (single or double)
industry = "Technology"
description = 'Leading software company'
location = "Tokyo, Japan"

# Integers - whole numbers
employees = 1000
founded_year = 2010
num_offices = 5

# Floats - numbers with decimals
rating = 4.7
salary = 4500000.00
growth_rate = 15.5

# Booleans - True or False (note capitalization!)
remote_work = True
hiring = False
has_benefits = True

Checking Data Types

# Use the type() function to check data types
company_name = "Sony"
employee_count = 50000

print(type(company_name))    # Output: <class 'str'>
print(type(employee_count))  # Output: <class 'int'>

Why Data Types Matter

Different operations work with different data types:

  • You can add numbers: 100 + 200 = 300
  • You can concatenate strings: "Hello" + " " + "World" = "Hello World"
  • But you can't add a string to a number: "100" + 200 will cause an error!

Lists and Dictionaries | リストと辞書

So far we've stored single values in variables. But what if we want to store multiple pieces of information together? That's where lists and dictionaries come in.

これまで、変数に単一の値を保存してきました。しかし、複数の情報をまとめて保存したい場合はどうでしょうか？

Lists: Collections of Items

A list is an ordered collection of items. Think of it as a numbered list where each item has a position (called an index).

# Creating lists - use square brackets []
companies = ["Sony", "Toyota", "Rakuten", "Nintendo"]
employee_counts = [50000, 370000, 28000, 6700]

# Accessing items by index (starts at 0!)
print(companies[0])      # Output: Sony (first item)
print(companies[3])      # Output: Nintendo (fourth item)

# Getting the length of a list
print(len(companies))  # Output: 4

# Adding items to a list
companies.append("Panasonic")
print(companies)      # Output: ["Sony", "Toyota", "Rakuten", "Nintendo", "Panasonic"]

Important: Python Uses Zero-Based Indexing

In Python (and most programming languages), counting starts at 0, not 1:

  • First item: index 0
  • Second item: index 1
  • Third item: index 2
  • And so on...

Dictionaries: Organized Key-Value Pairs

A dictionary stores data as key-value pairs. Instead of accessing items by position (like lists), you access them by key (a descriptive label).

# Creating dictionaries - use curly braces {}
company_info = {
    "name": "Sony Corporation",
    "industry": "Technology & Electronics",
    "employees": 50000,
    "location": "Tokyo",
    "is_hiring": True
}

# Accessing values by key
print(company_info["name"])        # Output: Sony Corporation
print(company_info["employees"])  # Output: 50000

# Adding new key-value pairs
company_info["founded_year"] = 1946

# Modifying existing values
company_info["employees"] = 51000

Lists vs. Dictionaries: When to Use Each

Use Lists When... Use Dictionaries When...
You have a collection of similar items You have related information about one thing
Order matters You want descriptive labels for each value
Example: ["Sony", "Toyota", "Rakuten"] Example: {"name": "Sony", "employees": 50000}
Storing multiple companies Storing detailed info about one company

Introducing Pandas | pandasの紹介

pandas is a Python library specifically designed for working with data. Think of it as "Excel in Python" - it lets you organize, analyze, and manipulate data in tables.

pandasはデータを扱うために特別に設計されたPythonライブラリです。

What is pandas?

pandas = Python's Data Analysis Tool

pandas provides two key data structures:

  • Series: A single column of data (like one column in Excel)
  • DataFrame: A table with rows and columns (like an entire Excel spreadsheet)

Why We Need pandas for ML

Task How pandas Helps
Loading Data Read CSV, Excel, JSON files easily
Viewing Data Display data in organized tables
Cleaning Data Handle missing values, remove duplicates
Analyzing Data Calculate statistics, filter, sort, group data
Preparing for ML Format data exactly how ML models need it

Basic pandas Operations

# First, import the pandas library
import pandas as pd

# Load data from a CSV file
companies = pd.read_csv("companies.csv")

# View the first 5 rows
print(companies.head())

# Check the shape (rows, columns)
print(companies.shape)        # Example output: (100, 5)

# See column names
print(companies.columns)

# Get summary statistics
print(companies.describe())

# Access a specific column
print(companies["company_name"])
