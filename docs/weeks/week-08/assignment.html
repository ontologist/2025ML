Activity 1: Missing Data Analysis (12 minutes)

アクティビティ1：欠損データ分析（12分）

What to Do:

  1. Load your Week 7 cleaned dataset in Google Colab
    Google Colabで第7週クリーニング済みデータセットをロード
  2. Calculate missing value statistics for each column
    各列の欠損値統計を計算
  3. Visualize missing data patterns using bar plots
    棒グラフを使用して欠損データパターンを可視化
  4. Analyze and categorize missing data by importance
    重要性で欠損データを分析および分類

Example Code / コード例:

# Count missing values per column
missing_counts = df.isnull().sum()
missing_percent = (df.isnull().sum() / len(df)) * 100

missing_summary = pd.DataFrame({
    'Missing_Count': missing_counts,
    'Missing_Percent': missing_percent
})
print(missing_summary[missing_summary['Missing_Count'] > 0])

# Visualize
missing_summary[missing_summary['Missing_Count'] > 0].plot(
    kind='bar', y='Missing_Percent'
)
plt.title('Missing Data by Column')
plt.ylabel('Percentage Missing')
plt.show()

Activity 2: Strategy Selection (10 minutes)

アクティビティ2：戦略選択（10分）

What to Do:

  1. For each column with missing data, decide on a strategy:
    欠損データを持つ各列について、戦略を決定：
    • Deletion: Remove rows/columns with missing critical data
    • Imputation: Fill with mean, median, mode, or fixed value
    • Flagging: Create indicator before imputing
  2. Document your rationale for each decision
    各決定の根拠を文書化
  3. Create a strategy plan table
    戦略計画表を作成

Decision Guidelines / 決定ガイドライン:

  • If <5% missing and not critical: delete rows
  • If numeric: consider mean/median imputation
  • If categorical: use mode or add "Unknown" category
  • If >30% missing: question if field is needed

Expected Output:

  • Strategy plan document with clear rationale
    明確な根拠を含む戦略計画文書

Activity 3: Implementing Missing Data Strategies (15 minutes)

アクティビティ3：欠損データ戦略の実装（15分）

What to Do:

  1. Implement your chosen strategies using Python
    Pythonを使用して選択した戦略を実装
  2. Apply deletion, imputation, and/or indicator creation
    削除、代入、および/またはインジケーター作成を適用
  3. Verify implementation by checking missing values after
    後で欠損値をチェックして実装を検証
  4. Document what was done for each field
    各フィールドで何が行われたかを文書化

Example Code / コード例:

# Deletion
df = df.dropna(subset=['company_name', 'industry'])

# Numeric imputation
df['employee_count'].fillna(df['employee_count'].median(), inplace=True)

# Categorical imputation
df['location'].fillna('Unknown', inplace=True)

# Create indicator
df['revenue_was_missing'] = df['revenue'].isnull().astype(int)
df['revenue'].fillna(df['revenue'].median(), inplace=True)

# Verify
print("Missing values after handling:")
print(df.isnull().sum())

Expected Output:

  • Dataset with missing data handled

  • Significantly reduced missing values

Activity 4: Outlier Detection and Treatment (13 minutes)

アクティビティ4：外れ値の検出と処理（13分）

What to Do:

  1. Identify numeric columns for outlier analysis
    外れ値分析のための数値列を特定
  2. Detect outliers using IQR method
    IQR方法を使用して外れ値を検出
  3. Visualize outliers using box plots
    ボックスプロットを使用して外れ値を可視化
  4. Decide treatment for each outlier (keep, remove, or cap)
    各外れ値の処理を決定（保持、削除、または制限）
  5. Implement treatments with justification
    正当化を伴う処理を実装

Example Code / コード例:

# Calculate IQR
Q1 = df['employee_count'].quantile(0.25)
Q3 = df['employee_count'].quantile(0.75)
IQR = Q3 - Q1

# Define outlier bounds
lower_bound = Q1 - 1.5 * IQR
upper_bound = Q3 + 1.5 * IQR

# Find outliers
outliers = df[(df['employee_count'] < lower_bound) |
              (df['employee_count'] > upper_bound)]
print(f"Found {len(outliers)} outliers")

# Visualize
df.boxplot(column='employee_count')
plt.title('Employee Count Distribution')
plt.show()

# Treatment example: remove clear errors
df = df[df['employee_count'] >= 0]  # Remove negatives
