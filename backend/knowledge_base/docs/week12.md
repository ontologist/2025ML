# 第12週：モデル評価と解釈

## Learning Objectives

By the end of this week, you will be able to:

- **Evaluate** model performance comprehensively
- **Interpret** model predictions and decisions
- **Analyze** feature importance
- **Communicate** model results effectively
- **Identify** model limitations and biases

学習目標：
- モデルパフォーマンスを包括的に評価する
- モデルの予測と決定を解釈する
- 特徴の重要性を分析する
- モデル結果を効果的に伝える
- モデルの制限とバイアスを特定する

---

## Topics Covered

### Comprehensive Evaluation
- **Multiple Metrics**: Accuracy, precision, recall, F1, AUC-ROC
- **Confusion Matrix Analysis**: Understanding errors
- **Learning Curves**: Detect overfitting/underfitting
- **ROC Curves**: Visualize trade-offs

### Model Interpretation
- **Feature Importance**: Which features matter most?
- **Coefficient Analysis**: For linear models
- **Decision Trees Visualization**: See decision rules
- **SHAP Values**: Explain individual predictions

### Error Analysis
- **Misclassification Analysis**: Which companies are hard to classify?
- **Bias Detection**: Are certain company types favored?
- **Edge Cases**: When does the model fail?
