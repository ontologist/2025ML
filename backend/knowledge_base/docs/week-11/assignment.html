Common Issues & Solutions / 一般的な問題と解決策

Problem: Platform won't accept CSV file

Solution: Open in Excel/Sheets and re-save as CSV. Ensure no special characters in column names. Check file size requirements. Try different browser.

Problem: Platform says dataset is too small

Solution: 20-25 companies should work for most platforms. If not, try a different platform. Some have minimum row requirements - check documentation. This is a learning exercise!

Problem: Training fails or returns error

Solution: Check for missing values in features. Verify all feature columns are numeric. Ensure label column has only 0 and 1. Review configuration - is target variable set correctly?

Problem: Training takes very long (>20 minutes)

Solution: Check internet connection. Platform may be busy - try off-peak hours. Consider using a different platform. For ~25 companies, should not take more than 15 minutes.

Problem: Training accuracy seems very low (e.g., 60%)

Solution: This is okay for first attempt! Random guessing = 50%, so 60% means model is learning. With small dataset, 60-80% is realistic. We'll evaluate properly next week. Don't worry yet!

Problem: Can't figure out how to save/export model

Solution: Look for "Save Model", "Export", or "Download" buttons. Check platform documentation. Some platforms auto-save to your account. Take screenshot of model ID/name. Ask instructor or use bot for help.

"During the training process, I was surprised by how quickly the platform processed my data. It took about 8 minutes total from upload to final results. The platform automatically selected Random Forest as the best algorithm for my dataset, achieving 72% accuracy on the test set.

I think the model learned to identify companies I find interesting based on patterns in industry type and company size. The accuracy of 72% is better than random guessing (50%), which means the model is definitely learning something from my labels. I'm curious to see how it performs on completely new companies next week.

Using the no-code platform was much easier than I expected. I didn't have to write any code, just upload my CSV and click through configuration screens. This experience showed me that ML is accessible even without programming expertise."
