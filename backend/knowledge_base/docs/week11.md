## Topics Covered

### Hyperparameter Tuning
- **Grid Search**: Exhaustive search over parameter grid
- **Random Search**: Random sampling of parameters
- **Bayesian Optimization**: Smart parameter search

### Regularization
- **L1 (Lasso)**: Feature selection through sparsity
- **L2 (Ridge)**: Prevent large coefficients
- **Elastic Net**: Combination of L1 and L2

### Cross-Validation
- **K-Fold CV**: Split data into k folds
- **Stratified K-Fold**: Maintain class distribution
- **Leave-One-Out**: Use each sample once for validation

### Model Selection
- Compare models using CV scores
- Consider training time vs performance
- Evaluate on holdout test set
