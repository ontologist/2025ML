## Week 14 Schedule

### Day 1-2: Final Presentations
**Format:** 10-15 minute presentations per student

**Presentation Structure:**
1. **Introduction** (2 min)
   - Problem: Why build a company recommendation system?
   - Personal motivation and goals

2. **Data & Methodology** (3 min)
   - Dataset description (size, sources, features)
   - Preprocessing approach
   - Feature engineering decisions

3. **Model & Results** (3 min)
   - Algorithms tested
   - Best performing model
   - Evaluation metrics and results

4. **Demo** (4 min)
   - Live system demonstration
   - Show recommendations for different user preferences
   - Explain how it works

5. **Reflection & Future Work** (2 min)
   - Challenges faced and overcome
   - Lessons learned
   - Ideas for improvement

6. **Q&A** (3-5 min)
   - Answer questions from classmates and instructor

### Day 3: Course Reflection & Celebration

**Activities:**
- Peer feedback and discussion
- Course retrospective
- ML learning path planning
- Certificate ceremony
- Celebration and networking

### 1. Presentation Slides
- **Format**: PDF or PowerPoint
- **Length**: 10-15 slides
- **Content**: Cover all required sections
- **Bilingual**: English or Japanese acceptable

### 2. Project Report
- **Format**: PDF document
- **Length**: 10-15 pages
- **Sections**:
  - Executive Summary
  - Introduction and Problem Statement
  - Data Collection and Description
  - Methodology (Preprocessing, Features, Models)
  - Results and Evaluation
  - Discussion and Analysis
  - Conclusion and Future Work
  - References
  - Appendices (code samples, additional charts)

- **Format**: ZIP file or GitHub repository
- **Contents**:
  - All Python scripts/notebooks
  - Saved models and vectorizers
  - README with setup instructions
  - Requirements.txt
  - Sample data (if shareable)
  - Documentation

### 4. Learning Reflection Essay
- **Format**: PDF or Word
- **Length**: 500-750 words
- **Topics**:

  - Most challenging aspects of the course
  - Most rewarding achievements
  - How will you apply ML skills in future?
